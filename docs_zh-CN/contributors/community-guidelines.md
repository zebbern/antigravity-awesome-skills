# 行为准则

## 我们的承诺

为了营造开放和友好的环境，我们作为贡献者和维护者承诺，让每个人在参与我们的项目和社区时都能获得无骚扰的体验。

## 我们的标准

以下行为有助于创造积极环境：

- 使用热情和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的方面
- 对其他社区成员表示同理心

参与者不可接受的行为包括：

- 使用性化语言或图像，以及不受欢迎的性关注或示好
- 恶意挑衅、侮辱/贬低性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人隐私信息
- 其他在专业环境中可能被认为不当的行为

## 执行

项目维护者负责明确可接受行为的标准，并预期对任何不可接受行为的实例采取适当和公平的纠正措施。

## 归属

## 报告指南

如果您遇到或目睹了不当行为，请按以下步骤报告：

### 1. 立即报告
- 发送邮件至：conduct@antigravity-skills.org
- 或创建GitHub Issue，标签：`conduct-report`

### 2. 报告内容
请包含以下信息：
- 发生时间
- 相关人员
- 具体行为描述
- 任何相关截图或证据

### 3. 处理流程
我们承诺：
- 24小时内确认收到报告
- 48小时内开始调查
- 保护报告者隐私
- 公平公正处理

## 冲突解决

### 小分歧
当遇到小分歧时：
1. **私下沟通** - 直接与相关人员联系
2. **理解对方** - 倾听不同观点
3. **寻求共识** - 找到双方同意的解决方案
4. **向前看** - 专注于项目目标

### 严重问题
当遇到严重问题时：
1. **联系维护者** - 通过官方渠道报告
2. **提供证据** - 具体事实和时间线
3. **保持专业** - 避免情绪化语言
4. **信任流程** - 让维护者处理

### 申诉流程
如果您对处理结果不满意：
1. **书面申诉** - 发送邮件至appeals@antigravity-skills.org
2. **包含详情** - 原始问题和处理结果
3. **说明理由** - 为什么认为处理不当
4. **等待审查** - 委员会将在7天内回复
